Efecto cascada con CSS

Cuando establecés una propiedad de un selector como 'p' con un cierto valor, ese valor se aplica a todas las etiquetas p. Sin embargo, si cambiás esa misma propiedad a un valor diferente para una instancia de p más específica, ese cambio sobreescribirá la 'regla general'.

  • Si decís: p { font-family: Garamond}, todas las 'p' tendrán fuente Garamond.

  • PERO si dices li p {font-family: Verdana}, las 'p' fuera de las 'li' tendrán fuente Garamond, y las 'p' DENTRO de las 'li' tendrán fuente Verdana.

  • Cuanto más específicos sean tus selectores, más importancia le da CSS al estilo que apliques.

Una mayor especificidad hace que CSS le dé prioridad a ese estilo particular.
